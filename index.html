Comprendre les fondations de l'Apprentissage Automatique

Pourquoi vous former avec ML Week ?

Notre objectif est d'offrir des formations vous apportant des compétences professionnelles dans le domaine de l'Apprentissage Automatique.

Nos formations sont avant tout pratiques.
A l'issue de ML Week, vous aurez :
* une bonne compréhension des fondamentaux
* une expérience d'utilisation de librairies utiles
* des exemples de code que vous aurez écrit pendant les formations

Les mathématiques vous font peur ?
Pas de panique, nous allons vous réconcilier avec les mathématiques en vous donnant des clés intuitives pour aider à mieux comprendre les concepts

Les travaux pratiques seront réalisés en Python (surtout scikit-learn) pour la partie algorithmique et Javascript (surtout D3) pour la partie visualisation.

Vous êtes fan de Python, Javascript ? Tant mieux, vous serez dans votre zone de confort.

Vous ne connaissez pas Python ou Javascript ? Ne vous inquiétez pas, tout est prévu pour que vous puissiez pleinement profiter des formations. Nous vous proposerons un tutoriel à exécuter avant les formations et nous ferons des rappels réguliers pendant.

A l'issue de ML Week, vous saurez comment amorcer un projet d' Apprentissage Automatique et surtout vous aurez la capacité de continuer votre apprentissage dans des domaines plus précis qui vous intéressent.
